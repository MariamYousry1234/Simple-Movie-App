# ?? Simple Movie App

A simple console-based Movie Management System built with **C++**.  
It allows you to manage a collection of movies stored in a text file and perform various operations on them.

## ? Features

- ? Add a new movie (name, year, rating)
- ?? Edit an existing movie
- ??? Delete a movie
- ?? Show all movies
- ?? Search for a movie by name
- ?? Filter movies by production year
- ? Filter movies by rating threshold
- ?? Sort movies by rating (ascending/descending)
- ?? Show highest-rated movies
- ?? Show lowest-rated movies
- ?? Show average rating
- ?? Show additional movie statistics
- ?? Display text-based rating categories (Terrible, Poor, Average, Good, Excellent)

## ??? Technologies Used

- **C++**
- File handling (text file storage)
- STL: `vector`, `queue`, `stack`
- OOP concepts: Classes, Enums, Static functions, Properties

## ?? How to Run

1. Download or clone the repository.
2. Open the project in :contentReference[oaicite:1]{index=1}.
3. Build and run the project (`Ctrl + F5`).
4. Follow the on-screen instructions to use the system.

## ?? Data Storage

- All movies are saved inside a file named `Movies.txt`.
- If the file does not exist, it will be created automatically after adding the first movie.
